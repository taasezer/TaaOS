# TaaOS Wallpaper Collection
# Rosso Corsa themed wallpapers

This directory contains high-resolution wallpapers for TaaOS:

1. taaos-default.png (3840x2160) - Main Rosso Corsa abstract design
2. taaos-dark.png (3840x2160) - Dark variant with red accents
3. taaos-minimal.png (3840x2160) - Minimalist logo on black
4. taaos-code.png (3840x2160) - Code-themed background
5. taaos-neural.png (3840x2160) - Neural network visualization

All wallpapers feature:
- Rosso Corsa (#D40000) primary color
- Deep black (#0A0A0A) background
- 4K resolution (3840x2160)
- Optimized for both light and dark themes
- Professional, developer-focused aesthetic

To generate actual wallpapers, use:
```bash
./generate-wallpapers.sh
```

This will create PNG files using ImageMagick with the TaaOS color scheme.
