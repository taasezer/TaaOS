# TaaOS Logo Assets
# Vector and raster logos for branding

This directory contains TaaOS logo files in various formats:

## Files:
- taaos-logo.svg (Vector, scalable)
- taaos-logo-1024.png (High-res PNG)
- taaos-logo-512.png (Medium-res PNG)
- taaos-logo-256.png (Standard PNG)
- taaos-logo-128.png (Small PNG)
- taaos-logo-64.png (Icon size)
- taaos-logo-white.svg (White variant for dark backgrounds)
- taaos-logo-red.svg (Rosso Corsa variant)

## Design Specifications:
- Primary Color: #D40000 (Rosso Corsa)
- Background: Transparent or #0A0A0A
- Font: Bold, modern sans-serif
- Style: Minimalist, tech-focused

## Usage:
- Boot splash: taaos-logo-512.png
- Desktop icon: taaos-logo-64.png
- Documentation: taaos-logo.svg
- Website: taaos-logo-1024.png

All logos maintain the TaaOS brand identity with Rosso Corsa theme.
